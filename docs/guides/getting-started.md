# Getting Started with NetworkSync

This guide walks you through setting up NetworkSync in your Godot C# project.

## Prerequisites

- **Godot 4.4+** with **.NET 8** support enabled
- A C# Godot project

## Installation

Copy `addons/networksync/` into your Godot project's `addons/` folder.

Enable the addon in **Project → Project Settings → Plugins**.
